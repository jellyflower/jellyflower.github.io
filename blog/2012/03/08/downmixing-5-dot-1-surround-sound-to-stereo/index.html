Downmixing 5.1 Surround Sound to Stereo

Mar 8th, 2012 6:53 pm

By far I think using FFmpeg is still the best way to do it. A command like this will finish the job:

ffmpeg -i INPUT -vn -ac 2 -acodec CODEC OUTPUT
